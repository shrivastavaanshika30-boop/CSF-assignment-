PROJECT OVERVIEW:-

This project is a static profile web page developed as part of the csf assignment. The webpage presents a simple and clean profile layout that includes structured sections such as personal information, introduction, and styled content. This project is created using HTML and CSS in VS code .The main objective of this project is to understand the fundamentals of web development , page structuring , styling , and version control using Git and GitHub.
